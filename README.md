# 实践基地标签分类系统

这是一个基于AI的实践基地标签分类系统，可以自动为实践基地生成三级标签体系。

## 功能特点

- ✅ 自动识别实践基地所在省份
- ✅ 智能判断习近平总书记是否到访
- ✅ 自动匹配相关主题（支持20+主题）
- ✅ 对应《概论》章节
- ✅ 支持多主题分类

## 使用方法

1. 输入实践基地名称（如：湖南十八洞村）
2. 系统自动分析并输出三级标签体系
3. 查看完整的分类结果

## 支持的主题

包括但不限于：攀高攻坚、大国重器、绿水青山、健康中国、中华文脉、乡土中国、红色记忆、脱贫攻坚乡村振兴、科学精神、工业文化、美丽中国等。

## 技术栈

- LangChain + LangGraph
- DeepSeek R1 模型
- Streamlit
- Web Search SDK
